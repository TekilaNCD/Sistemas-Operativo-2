Simulador de Planificación de Procesos – SIGET

Este repositorio contiene el desarrollo de un simulador de planificación de procesos inspirado en el funcionamiento de los sistemas operativos y aplicado al contexto del SIGET (Sistema de Gestión de Tráfico).

En este proyecto se implementa una simulación de procesos con diferentes características como tiempo de ejecución, prioridad de alerta y tamaño de datos, permitiendo observar cómo son gestionados por el sistema.

El simulador muestra los estados de los procesos (Nuevo, Listo, En ejecución y Terminado) y aplica dos algoritmos de planificación de CPU:

FIFO (First In First Out)

Round Robin

El objetivo es analizar cómo cada algoritmo gestiona múltiples tareas y cuál ofrece un mejor equilibrio entre eficiencia y tiempo de respuesta en el procesamiento de datos del sistema SIGET.
